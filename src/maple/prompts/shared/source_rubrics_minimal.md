## Source Guidelines

**Primary sources:** Original experimental data with real DOIs.
**Secondary sources:** Reference values, textbooks. Use `doi_or_url` field.

## Text Snippets (CRITICAL)

Snippets are automatically verified against full paper text:

- **VERBATIM only** - copy exact text, never paraphrase
- **No table reconstruction** - don't create `CD8^{+} | ... | 17` notation
- **Short continuous spans** - 5-50 words containing the value
- **Plain text** - write `CD8+` not `CD8^{+}`

**Good:** `"median CD8+ density was 17 (IQR 9-30) cells/HPF"`
**Bad:** `"CD8^{+} | No neoadjuvant | 17 (9-30)"` (reconstructed)

**Figure-extracted values:** Describe figure and reading, e.g., "Figure 2A bar height ~45 (estimated from axis)"
